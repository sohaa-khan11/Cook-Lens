# CookLens | The Future of Culinary Vision

CookLens is a premium, cinematic cooking companion that transforms how you interact with ingredients. Using advanced AR scanning and AI synthesis, it turns your pantry into a collection of chef-grade culinary experiences.

## 🌟 Product Vision

CookLens is designed to be more than a recipe app. It is an immersive, editorial experience that celebrates the craft of cooking.
- **Cinematic Desktop**: A high-impact storytelling layer for product discovery.
- **Native Mobile App**: A high-tactile, performant dashboard for real-time kitchen mastery.

## 🚀 Key Features

- **Neural AR Scan**: Real-time ingredient recognition with a cinematic HUD.
- **Digital Vault**: A secure, organized repository for your discovered recipes and "Synthesis" results.
- **Synthesis Engine**: Intelligent recipe generation based on identified pantry items.
- **Editorial Experience**: High-end typography and motion design that feels like a premium digital magazine.

## 🛠️ Tech Stack

- **Framework**: Next.js 16 (React) with TypeScript
- **Motion**: Framer Motion (Cinematic easing, parallax, scroll-based reveals)
- **Styling**: Vanilla CSS with cinematic utility tokens (Glassmorphism, High-contrast dark mode)
- **State Management**: React Hooks & Motion MotionValues for interaction-driven UI.

## 📦 Getting Started

### Prerequisites

- Node.js 18+
- npm / yarn

### Installation

1. Clone the repository:
   ```bash
   git clone <repository-url>
   cd CookLens/frontend
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Run the development server:
   ```bash
   npm run dev
   ```

## 📐 Project Structure

- `src/app/`: File-based routing and page components.
- `src/components/`: Reusable cinematic UI tokens and layout elements.
- `src/app/scan/`: AR HUD and real-time recognition layer.
- `src/app/vault/`: Recipe management and list views.
